I've been meaning to build a personal site for years. Every time I started, I'd get lost in the tooling — which framework, which CMS, which host — and by the time the scaffolding was done I'd lost interest in writing anything.

So this time I skipped the ceremony. Next.js, MDX, Tailwind, done. No CMS, no database, no clever tricks. Every page is a file, every note is a file, every photo is an entry in a list. When I want to write something, I open the editor and write.

## What this site is

Five sections, four of them quiet:

- **Home** — where you land.
- **About** — a slightly longer introduction than a Twitter bio.
- **Now** — what I'm actually working on, reading, and thinking about this month.
- **Notes** — rough writing. Not a blog. Not a newsletter. Just notes.
- **Photos, Skincare, Supplements** — side collections that I kept adding to, so they get their own pages.

## What this site isn't

It's not a portfolio. It's not optimized for recruiters. There's no contact form. If you want to reach me, email is at the top of the [links](/links) page.

## Why a site at all

Because social platforms come and go. Because I want a place that's mine, that doesn't disappear when some company changes its API or shuts down. Because long-form thinking is better than short-form thinking, and because the act of publishing — even if no one reads it — makes you write better than leaving things in drafts.
